# DriveHub - Full-Stack Car Dealership

## Tech Stack

- **Frontend:** Next.js 14, Tailwind CSS, TypeScript
- **Backend:** Node.js, Express.js, TypeScript
- **Database:** PostgreSQL with Prisma ORM
- **Auth:** JWT (access + refresh tokens)
- **Payments:** Stripe

## Setup

### Prerequisites

- Node.js 18+
- PostgreSQL 14+

### Installation

```bash
# Clone the repo
git clone <repo-url>
cd car-dealer

# Install server dependencies
cd server
npm install

# Install client dependencies
cd ../client
npm install
```
